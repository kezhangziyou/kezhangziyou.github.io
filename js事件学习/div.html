div可定义文档中的分区或节（division/section）。 div标签可以把文档分割为独立的、不同的部分。它可以用作严格的组织工具，并且不使用任何格式与其联。 如果用 id 或 class 来标记 div，那么该标签的作用会变得更加有效。用法div 是一个块级元素。 这意味着它的内容自动地开始一个新行。实际上，换行是 div固有的唯一格式表现。可以通过 div 的 class或 id 应用额外的样式。不必为每一个 div 都加上类或 id，虽然这样做也有一定的好处。 可以对同一个 div 元素应用 class 或 id 属性，但是更常见的情况是只应用其中一种。这两者的主要 差异 是，class 用于元素组（类似的元素，或者可以理解为某一类元素），而 id 用于标识单独的唯一的元素。
